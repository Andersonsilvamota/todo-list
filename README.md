##To run the project on your local machine:

1 - Clone the project:
  git clone https://github.com/Andersonsilvamota/todo-list.git
  
2 - Access the folder:
  cd todo-list

3 - Install the dependencies:
  npm i
  
4- Start the project:
  npm run dev
